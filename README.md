# Video Game Sales Analysis

## Objective
To analyze global video game sales using Exploratory Data Analysis (EDA).

## Dataset
Video Game Sales dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## EDA Performed
- Data cleaning and preprocessing
- Descriptive statistics (df.describe)
- Aggregation using groupby and pivot tables
- Year-wise, genre-wise, platform-wise and region-wise analysis
- Visualization with insights

## Key Insights
- Action genre has the highest global sales
- North America contributes the most sales
- Sales peaked around the mid-2000s

## Conclusion
EDA helped identify trends and patterns in the global video game industry.
